保姆级使用教程

从零开始，5 分钟运行属于自己的 AI 客户端

01

下载项目源码

点击下方按钮前往 GitHub，点击绿色的 "Code" 按钮，选择 "Download ZIP" 下载到本地，解压到任意文件夹。

前往 GitHub 下载源码
02

安装 Node.js 环境

Aether 需要 Node.js 18 或更高版本。如果尚未安装，请访问官网下载安装包（LTS 版本）。

下载 Node.js 安装后重启电脑，打开 CMD 输入 node -v 验证
03

安装项目依赖

在项目文件夹地址栏输入 cmd 回车打开命令行，然后粘贴以下命令：

npm install

等待进度条走完（约 1-3 分钟，根据网络状况），出现 added xxx packages 即成功。

04

启动开发服务器

依赖安装完成后，运行以下命令启动：

npm run dev

看到 Local: http://localhost:5173 后，按住 Ctrl 点击链接，或浏览器访问该地址。

05

配置 API 密钥

首次打开后，点击界面右上角的设置图标（⚙️）：

  • 输入你的 API 端点（如 https://api.openai.com/v1）
  • 粘贴你的 API Key（格式如 sk-...）
  • 点击"获取模型列表"，选择你要使用的模型
  • 关闭设置，开始对话！
提示： API Key 仅保存在你的电脑本地，采用 AES-GCM 加密，不会上传到任何服务器。
06

Windows 一键启动（可选）

如果你使用的是 Windows 系统，且已安装 Node.js，可以直接双击项目文件夹中的 Start Aether.bat 文件自动完成安装和启动，无需手动输入命令。

常见问题

Q: 提示 "npm 不是内部或外部命令" 怎么办？ A: 说明 Node.js 安装未完成或环境变量未生效，请重启电脑后重试，或重新安装 Node.js 并勾选 "Add to PATH" 选项。
Q: 安装依赖时卡住不动？ A: 可能是网络问题，尝试更换 npm 源：npm config set registry https://registry.npmmirror.com 后再运行安装命令。
Q: 如何获取 OpenAI API Key？ A: 访问 platform.openai.com，注册账号后在 "API Keys" 页面创建。注意：需要海外支付方式充值才能使用。
