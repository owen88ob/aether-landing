流式响应

实时打字机效果输出，无需等待完整响应，思考过程一目了然。

本地加密

AES-GCM 标准加密保护 API 密钥，所有敏感数据仅存于本地。

多模型切换

支持 OpenAI、Claude、Gemini 等多种 API，自动获取可用模型列表。

图像生成

集成 DALL-E 与 Stable Diffusion，文生图、图生图一站式完成。

文件解析

支持 PDF、图片、文本文件拖拽上传，自动提取内容并对话。

极简设计

深色玻璃拟态界面，60fps 流畅动画，为长时间使用优化。

现代化技术架构

基于 Vite + React + TypeScript 构建，轻量快速。支持 Windows、macOS 和 Web 端无缝切换。

  • TypeScript 全类型安全
  • Tailwind CSS 原子化样式
  • LocalStorage 本地持久化
  • SSE 流式数据传输
